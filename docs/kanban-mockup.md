# Content Pipeline Kanban Redesign

## Overview

This mockup proposes a consolidated Kanban board view that streamlines the content creation workflow. The key changes:

1. **Braindumps become raw input** - Quick capture, then processed into themes
2. **Themes are enhanced** - Rich markdown with post ideas, narrative arcs, potential post count
3. **Single Kanban view** - All stages visible at once for better pipeline visibility
